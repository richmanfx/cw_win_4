# CW_WIN_4

Эта программа предназначена для тех, кто изучает телеграф (CW, Азбуку Морзе). 
С помощью программы можно воспроизводить в случайном порядке слова из 
текстового файла.

Отличительной чертой программы является возможность при высокой скорости
устанавливать длительную паузу между словами -  настройка "Интервал", что 
даёт возможность воспринимать и осознавать всё слово целиком, а не отдельно 
по буквам.

Буквы, составляющие слова, могут быть как латинские, так и русские.
Кодировка файлов с рускими буквами должна быть CP-1251 (Windows-1251).
Знаки пунктуации можно воспроизводить как в русской, так и в латинской 
интерпретации.

Дополнительную информацию можно получить из файла win-помощи cw_win_4.chm или
в формате PDF cw_win_4.pdf.

- Можно сохранить выведенные на экран слова в текстовый файл, кодировка при 
  использовании русских букв будет CP-1251 (Windows-1251).
